X Window System (Servidor X ou X11)

É um protocolo de rede e vídeo que tem a capacidade de trabalhar com o sistema de janelas e que permite uma melhor interação por meio de teclado e mouse.

Fornece meios para o desenvolvimento de interfaces gráficas para usuários ou GUI (Graphical User Interface).

Sistemas de Interconexão de Interfaces

Os principais softwares de compartilhamento de interface gráfica são:

  • AnyDesk

  • TeamViewer

Ambos programas utilizam uma conexão SSH para conectar os dispositivos. O usuário a quem compartilha sua tela está efetuando um processo de upload para o usuário na qual recebe esses dados, onde por sua vez, está efetuando o download.

SSH (Secure Shell)

O Secure Shell ou Security Service Host é um protocolo de rede criptográfico para operação de serviços de rede de forma segura sobre uma rede insegura. O melhor exemplo de aplicação conhecido é para login remoto de usuários a sistemas de computadores.

Ele permite a você acessar virtualmente o servidor como se você estivesse em um terminal (no prompt do DOS, por exemplo). Se você preferir, considere como o SSH como um computador controlando outro computador.

Rede simétricas e assimétricas

Rede simétrica

Quantidade de Upload e Download são simétricas, possuindo a mesma velocidade de transmissão e recepção de dados.

Rede assimétrica

Quantidade de Upload e Download assimétricas, muito utilizado para usuários comuns, onde a velocidade de transmissão e recepção são diferentes.

Instalação e Uso da GUI

  • XORG ou X.Org Server é a implementação livre e de código aberto de servidor de exibição para o sistema de janelas X11 gerenciado pela X.Org Foundation, que é hospedada pela freedesktop.org, e concede acesso público ao padrão das liberações do X Window para os esforços da comunidade de software livre e de código aberto:

    $ sudo apt install xorg



    Para iniciar o software de interface gráfica X.Org digite o comando:

    $ tasksel



    Esse comando irá listar uma lista de interfaces gráficas disponíveis como recurso para o usuário instalar em seu sistema. Alguns exemplos de GUI:

    • GNOME
    • Xfce
    • KDE
    • Cinnamon
    • ...
